Full Deezer catalog
90M+ tracks. Search tracks, albums, artists, and playlists — all from Deezer's completely free, no-auth public API.
FLAC / Lossless streams
Streams are resolved via TidalFi (the same backend SpotiFLAC uses) using the track's ISRC code.
No account needed
Deezer's public API requires zero auth. No login, no API key, no credentials — it just works out of the box.
ISRC enrichment
Every track carries its ISRC code. Eclipse uses it to pull synced lyrics, artwork, and genre tags from Apple Music automatically.
Full catalog browsing
Tap into any album, artist, or playlist returned by search — Eclipse loads full track lists directly from this addon.
Offline downloads
Tracks with direct stream URLs can be saved for offline listening inside Eclipse. No server needed for playback once downloaded.
